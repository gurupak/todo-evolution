# Todo Frontend

Next.js 16 frontend for Phase II Evolution of Todo.

## Setup

```bash
npm install
cp .env.local.example .env.local
# Edit .env.local with your configuration
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Better Auth
- TanStack Query
